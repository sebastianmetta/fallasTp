# fallas_tp

Repositorio para 75.67 - Fallas 1 - 1erCuat 2015. Grupo: Maxi Roitman - Seba Metta. 
Tp: Sistema experto para el diagnóstico de posibles causas de fallas en equipos de refrigeración.

El presente proyecto implementa el sistema experto.

Toma los datos de la falla del usuario y como salida, otorga las posibles causas de la falla basado en la experiencia del experto.

## Usage

Para la ejecución del sistema, deberá contar con:
•	Plataforma Java compatible 
•	Java 1.7 o superior (JRE)
•	El archivo ejecutable proporcionado por el equipo de desarrollo.
Instrucciones de ejecución:
1.	Copie el archivo (jar) a un directorio sobre el cual tenga permisos de lectura y escritura.
2.	Abra una terminal y diríjase al directorio 1.
3.	Ejecute el comando: $ java -jar fallas_tp-0.1.0-standalone.jar
