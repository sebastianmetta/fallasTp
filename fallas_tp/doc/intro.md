# Introduction to fallas_tp

Para el desarrollo del sistema se utilizó como lenguaje de programación Clojure 1.6
Se almacenaron las reglas en una estructura interna. La evaluación de las mismas se realiza con los datos de entrada que proporciona el usuario, a saber: Tipo de equipo de refrigeración, Tipo de falla y Falla.
El sistema como salida otorga un listado con las posibles causas, descripción y soluciones a la falla indicada.
